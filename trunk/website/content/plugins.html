IntelliJ® IDEA

JBehave has an IDEA plug-in that allows the developers to launch behaviour verifications.

It can be downloaded through IDEA's plugin download site: http://www.jetbrains.com/idea/plugins/plugins.html

Eclipse

JBehave plugin for Eclipse allows you to launch the behaviour verification process through a short-cut a predefined run configuration.

You can use Eclipse' update mechanism by adding a "Remote Site" in your Eclipse Software Update configuration which links to:

http://www.jbehave.org/eclipse

After a successful installation, you should be able to see JBehave feature in your Eclipse about dialog
