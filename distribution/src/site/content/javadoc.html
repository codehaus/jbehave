JBehave Core

JBehave Core contains the fundamental functionality of JBehave

JBehave Extensions

JBehave Pico adds support for dependency injection using PicoContainer

JBehave Spring adds support for dependency injection using Spring Framework

JBehave Guice adds support for dependency injection using Guice

JBehave Plugins

JBehave Ant contains the Ant task to run JBehave scenarios.

JBehave Maven Plugin contains the Maven plugin to run JBehave scenarios.
