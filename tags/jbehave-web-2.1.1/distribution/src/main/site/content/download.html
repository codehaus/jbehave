Stable releases

The stable releases of JBehave Web can be found on the Codehaus Repository:

  • 2.1.1 bin distribution
  • 2.1.1 src distribution

Latest Snapshot

The snapshot builds of JBehave Web can be found on the Codehaus Snapshots Repository. Be sure to choose the snapshot with the latest 2.x version and the filename with the latest timestamp.

Using Maven Artifacts

If you use Maven as the build system (or one that is compatible with the Maven Repository, such as Ivy), you can declare JBehave Web dependencies as

   
    org.jbehave.web
    jbehave-web-runner
    [version]

specifying the correct version.
