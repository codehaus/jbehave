-----------------------------------------------
Using JMock with JBehave
-----------------------------------------------
h1(FirstChild). One minute tutorial

Create a Behaviour Class that extends <tt>UsingJMock</tt> from the jmock extension library.
<tt>UsingJMock</tt> brings together a bunch of "sugar" methods for use when creating/setting up
mocks and expectations. It also provides auto-verification for all mocks.

You use jMock like so:

<div class="Shaded"><pre>
	public class BarBehaviour extends UsingJMock {
	    public void shouldRequireTheUseOfMocks() {
	        // given
	        Mock cheese = new Mock(Cheese.class);
	        Sheep sheep = new Sheep((Cheese)cheese.proxy());

	        //expect
	        cheese.expects(once()).method("squelch").withAnyArguments();

	        // when
	        sheep.eatCheese();

	        // then
	        verify();
	    }
	}
</pre></div>

You can run your behaviour like so:
<div class="Shaded">
    	<pre>java -cp org.jbehave.jar;jmock.jar;your_classpath your.ApplicationBehaviour</pre>
</div>
